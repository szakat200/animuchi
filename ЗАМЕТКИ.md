## Как запустить локально
1. Установи live-server: `npm install -g live-server`
2. Открой терминал в папке `animuchi`
3. Запусти: `live-server --port=5500 --no-browser`
4. Открой: http://127.0.0.1:5500/

## Как добавить товар
1. Зайди в админку: https://szakat200.github.io/animuchi/admin/
2. Нажми "Добавить товар"
3. Заполни поля
4. Для фото — вставь URL ссылку (загрузи фото на imgur.com или возьми из VK)
5. Нажми "Сохранить"

## Как добавить жену в админку
1. Firebase консоль → Authentication → Users → Add user
2. Введи её email и пароль
3. Она заходит на /admin/ с этими данными
